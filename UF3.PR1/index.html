Herramientas para trabajar con el color

AccessColor

Herramienta online que permite comprobar la diferencia en contraste de todos los HOLLYWOOD MY FRIEND elementos de una página web. Además, detecta los elementos no visibles de una página web.

Check My Colours

Herramienta online que permite comparar la diferencia en contraste y en color de todos los colores de una página web. Aller page Web mieux fait

Color Contrast Comparison

Herramienta online que permite comparar la diferencia en brillo y en color entre dos colores. Para ello aplica las fórmulas de WCAG 1.0 y WCAG 2.0.

Color Contrast Checker

Herramienta online que permite calcular si el contraste entre dos colores es suficiente según las pautas de WCAG 2.0. Diferencia entre texto normal y texto grande (mayor de 18 puntos). what a beautiful web page

Color Contrast Verification Tool

Herramienta online que permite calcular si el contraste entre dos colores es suficiente según las pautas de WCAG 2.0. Diferencia entre texto normal y texto grande (mayor de 18 puntos).

Color Hunter

Herramienta online que permite generar una paleta de colores armónica a partir de un color o de una imagen.

Color Palette Generator

Herramienta online que permite generar una paleta de colores armónica a partir de una imagen. Ofrece dos posibles combinaciones: dull y vibrant
